# PMP Exam Prep 2026: Comprehensive Study Guide

[![Live Site](https://img.shields.io/badge/Live-Site-purple?style=flat&logo=vitepress)](https://dustinober.github.io/PMP-2026/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains a comprehensive, interactive study guide for the Project Management Professional (PMP)® exam, updated for the 2026 Examination Content Outline (ECO). Built with [VitePress](https://vitepress.dev/), it features 10 chapters covering all three domains (People, Process, Business Environment) plus practice exams and AI essentials.

**Live Site:** [dustinober.github.io/PMP-2026](https://dustinober.github.io/PMP-2026/)

## What's New in the 2026 Exam

| Change | Details |
|--------|---------|
| **Questions** | 185 questions (up from 180) |
| **Duration** | 240 minutes with 2 scheduled breaks |
| **AI Focus** | Enhanced coverage of AI & data analytics |
| **Sustainability** | Increased emphasis on ESG and sustainability |
| **Business Environment** | Increased from 8% to 26% of exam content |

## Exam Domain Breakdown (2026 ECO)

| Domain | Percentage | Focus Areas |
|--------|------------|-------------|
| **People** | 33% | Team leadership, conflict management, stakeholder engagement |
| **Process** | 41% | Scope, schedule, cost, risk, quality, procurement |
| **Business Environment** | 26% | Strategy, compliance, benefits realization, change management |

## Getting Started

### Development

```bash
# Install dependencies
npm install

# Start local development server
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```

The site will be available at `http://localhost:5173` during development.

### Deployment

The site is automatically deployed to GitHub Pages via the `deploy.yml` workflow. To deploy manually:

```bash
npm run deploy
```

This builds the site and pushes to the `gh-pages` branch, which serves the site at [dustinober.github.io/PMP-2026](https://dustinober.github.io/PMP-2026/).

## Project Features

- **Interactive Components** – Vue.js components for quizzes, charts, and visualizations
- **Knowledge Checks** – Built-in assessments at the end of each section
- **Progress Tracking** – Visual progress bars for learning modules
- **Responsive Design** – Optimized for desktop, tablet, and mobile
- **Dark Mode** – Automatic theme switching support
- **Full-Text Search** – Local search powered by VitePress

## Automation

GitHub workflows under `.github/workflows/` and command configs in `.github/commands/` support AI-assisted triage, review, and issue handling:

| Workflow | Purpose |
|----------|---------|
| `deploy.yml` | Automated deployment to GitHub Pages |
| `gemini-triage.yml` | AI-powered issue triage |
| `gemini-review.yml` | Automated code review assistance |
| `gemini-scheduled-triage.yml` | Periodic repository maintenance |
